Included in this release are the following files and directories:

bin/bossam.jar            : Bossam Inference Engine
bin/antlr.jar             : ANTLR Runtime
bin/rio.jar
bin/openrdf-util.jar
bin/openrdf-model.jar     : OpenRdf RIO Toolkit
bin/Piccolo.jar           : Piccolo SAX Parser
bin/nsam.jar              : NSAM N3 Parser
bin/bossam.sh             : Command-line Bossam Runner Script for Linux/Unix
bin/bossam.bat            : Command-line Bossam Runner Script for Windows
bin/bossamsh.sh           : Bossam Shell Runner Script for Linux/Unix
bin/bossamsh.bat          : Bossam Shell Runner Script for Windows
documentation/*           : A set of documents
samples/*                 : A set of sample rulebases


The latest documentation for Bossam can be found online at:

        http://mknows.etri.re.kr/bossam/
